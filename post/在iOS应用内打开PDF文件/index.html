在iOS应用内打开PDF文件

Jan 7, 2016, 5:13pm by zephan

  作者：Arthur Knopper 原文链接

  在iOS应用内显示PDF文件内容可以使用Web View来解决，这篇文章将从本地读取PDF文件并显示到屏幕上。教程的代码使用Xcode7.1、iOS9编译运行。

  打开Xcode新建一个新的名字为“IOS9OpenPDFTutorial”的项目，语言选择Swift，设备选择iPhone，其他随意。

  之后打开Main.storyboard，添加一个名字为“Open PDF”的按钮。再按住Ctrl键的同时点击按钮并拖动至父视图的顶部，松开点击（Ctrl键不松开）在出来的界面中选择“Vertical Spacing to Top Layout Guide” 和 “Center Horizontally in Container”选项。

界面看起来是这个样子。

为按钮链接点击事件到 ViewController.swift

这里有个PDF，下载之后加入到项目里面，然后下面是点击之后实现打开PDF的代码。

@IBAction func openPDF(sender: AnyObject) {
    let url = NSBundle.mainBundle().URLForResource("Alice_In_Wonderland", withExtension: "pdf")

    if let url = url {
      let webView = UIWebView(frame: self.view.frame)
      let urlRequest = NSURLRequest(URL: url)
      webView.loadRequest(urlRequest)

      view.addSubview(webView)
    }
  }

  这段代码从本地加载PDF文件得到地址，然后通过Web View加载。运行之后点击 “Open PDF” 按钮就可以看到PDF文件的内容了。

  你可以从这里(github)下载到“IOS9OpenPDFTutorial”的源代码（点击链接之后command+F搜索“IOS9OpenPDFTutorial”）
